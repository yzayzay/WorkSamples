Read Me!

***** Video Rental Store. *****

This application is based on a video store rental managemnt system using Java JDBC. The application tracks 
stores information on the following: videos rented out, videos available, movies by date, movies by genre,
movies by year. All the movie records are stored and retreived from the database. 
To run this application, you will need to setup the following programs. 

Download Eclipse
1.Download Java SE/JDK and choose the one that mataches your systems requirement (32bit or 64bit or Mac)
  http://www.oracle.com/technetwork/java/javase/downloads/index.html
  Following the installation wizard to completion. 
1. Download eclipse from below link and choose one that meets your system's requirement. 
   https://www.eclipse.org/downloads/packages/eclipse-ide-java-developers/keplersr1 
   Following the installation wizard to completion.
2. Download V1 Release from github or commit a new pull request for all files. 
   Make sure to save the file in a memoriable location for easy access. 
3.Open Eclipse and import V1 Release. 
   For information on how to import and export projects into java, see:
   http://realsearchgroup.org/SEMaterials/tutorials/import_export/ 
4. Once the project is imported makesure you have JDBC jar file 
   Select the apache Derby install with a recent release
   http://db.apache.org/derby/
5. Configure Apache Derby in Eclipse
   https://db.apache.org/derby/integrate/plugin_help/start_toc.html
6. After configuring Apache Derby run the VideoStoreTest.java file
